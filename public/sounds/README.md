# Sonidos del juego

Coloca aquí los archivos de sonido:
- **drop1.mp3** - Sonido para la primera colisión de cada burbuja (opción 1)
- **drop2.mp3** - Sonido para la primera colisión de cada burbuja (opción 2)
- **pop1.mp3** - Sonido de fusión (opción 1)
- **pop2.mp3** - Sonido de fusión (opción 2)

Los sonidos deben ser cortos (menos de 1 segundo) y con formato MP3 para mejor compatibilidad.

## Comportamiento de sonidos:
- `drop1.mp3` y `drop2.mp3`: Se reproducen aleatoriamente cuando una burbuja toca cualquier superficie por primera vez
- `pop1.mp3` y `pop2.mp3`: Se reproducen aleatoriamente cuando dos burbujas se fusionan

Puedes encontrar sonidos gratuitos en:
- freesound.org
- zapsplat.com (requiere registro)
- pixabay.com/sound-effects/
